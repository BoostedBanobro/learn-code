# About

Learn Code is a website that can help you learn or teach the basics of FTC programming, brought to you by FTC 9656 Omega!

## General Topics Covered

-   Java
-   Command Line
-   Git and GitHub
-   FTC SDK
-   Roadrunner
-   FTC CV
-   Control Theory

For a more detailed breakdown, see [Units](/learn-code/units).
